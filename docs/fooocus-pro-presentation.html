01 · Вступ

Що таке Fooocus і як це працює

Fooocus — безкоштовна open-source програма для генерації зображень з тексту. Ти описуєш сцену словами → нейромережа «малює» фото → отримуєш PNG. Інтерфейс у браузері, без Photoshop і без підписки на Midjourney.

На чому це побудовано

Движок (AI)Stable Diffusion XL (SDXL) — велика нейромережа, навчена на мільйонах фото. «Розуміє» англійський текст і перетворює його на пікселі.
Модель у ColabJuggernaut XL Ragnarok — варіант SDXL, заточений під фотореалізм (шкіра, світло, тіло).
LoRAНевеликі «доповнення» до моделі (anatomy, detail) — підсилюють окремі аспекти без заміни базової моделі.
ІнтерфейсGradio — веб-UI у браузері (поля, кнопки, галерея).
Де рахуєGPU (відеокарта). У нас — Google Colab T4 в хмарі; на комп’ютері нічого не встановлюєш.
КодБаза — lllyasviel/Fooocus. Наш форк — sunsideaspect/foocus_sunside (Colab, preset, без blur).

Чим не схоже на ChatGPT Images

  • ChatGPT — «чорна скринька», мало налаштувань
  • Fooocus — повний контроль: розмір, seed, стилі, inpaint, FaceSwap
  • Детальний промпт = передбачуваніший результат

Fooocus Pro (наш Colab)

  • Один ноутбук → Run all → лінк на UI
  • Preset Juggernaut + anatomy LoRA вже налаштовані
  • Без автоматичного blur на контенті
  • Оптимізовано під безкоштовний T4
Аналогія: SDXL — це «художник», Prompt — технічне завдання, LoRA/Styles — референси стилю, GPU — час роботи художника, Gallery — готове полотно.
02 · Запуск

Як потрапити в інтерфейс

  1. Відкрий Colab-ноутбук
  2. Runtime → Change runtime type → GPU (T4)
  3. Runtime → Restart session
  4. Runtime → Run all
  5. Дочекайся Open Fooocus (Colab proxy) → нова вкладка
Перший запуск: 10–15 хв (моделі). Наступні картинки: 2–5 хв.

05 · Перемикачі

Три галочки над Input Image

Input ImageВідкриває панель роботи з готовим фото: upscale, референс, inpaint, describe, metadata.
EnhanceДругий етап після генерації: авто-дорисовка face / hand / body по маці (SAM + inpaint).
AdvancedПрава колонка: Settings, Styles, Models, Advanced — розмір, стилі, моделі, кроки.
Перший тест: Input Image OFF, Enhance OFF, Advanced ON.
06 · Settings

Settings (права колонка)

PerformanceQuality (~60 кроків) — детальніше. Speed (~30) — швидше. Lightning/Hyper — дуже швидко, гірша якість.
Aspect RatiosПропорції: 768×1344 portrait, 1024×1024 square, інші — landscape / full body.
Image NumberСкільки варіантів за один Generate. 2–4 = вибрати найкращий кадр.
Output FormatPNG (без втрат), JPEG, WebP.
Negative PromptЧого не хочеш: cartoon, blur, bad anatomy… (у Colab заповнено preset-ом).
Random / SeedRandom ON = кожна генерація нова. Random OFF + seed = повторюваний результат.
History LogПосилання на локальний лог згенерованих файлів.
07 · Styles

Styles

ПошукФільтр списку стилів.
CheckboxGroupГотові «фільтри камери» — додають до промпту слова про освітлення, mood, тип зйомки.

Стилі не замінюють промпт — доповнюють його.

Для реалістичного phone-look: MRE Spontaneous Picture + Semi Realistic. Не змішуй Photo Iphone з cinematic-стилями.
08 · Models

Models

Base ModelОсновна модель. Colab: Juggernaut XL Ragnarok — фотореалізм.
RefinerДругий прохід для деталей. Рекомендовано None. Та сама модель = refiner ігнорується.
Refiner Switch At% кроків для перемикання на refiner (лише якщо refiner ≠ None).
LoRA 1–5Надбудови: anatomy, add detail (увімкнені в preset). Enable/Weight — сила ефекту.
RefreshОновити списки файлів після ручного додавання моделей.
Якщо обличчя слабке — знизь dynamic_anatomy Weight до 1.0–1.2 (замість 1.4).
09 · Advanced

Advanced + Debug Tools

Guidance Scale (CFG)Наскільки модель слухається промпту. 3.5–4.5 — реалізм. Preset: ~4.2.
Image SharpnessРізкість текстур. Занадто високо — «перешарп».
Developer Debug ModeВідкриває Debug Tools: кроки, sampler, scheduler.
Sampling StepКількість кроків генерації. Preset: 48. Sweet spot: 48–58. Не гони 80+ — «пластикова» шкіра.
Sampler / Schedulerdpmpp_2m_sde_gpu + karras — не міняти без причини.
Save MetadataЗберігає параметри в PNG → повтор через Metadata tab.
10 · Input Image

Input Image → Upscale & Image Prompt

Upscale or Variation

ImageЗавантаж своє або згенероване фото.
Vary Subtle / StrongВаріація схожого кадру без нового промпту.
Upscale 1.5x / 2xЗбільшення з дорисовкою деталей.

Image Prompt (до 4 слотів)

ImagePromptСтиль / композиція з референсу.
FaceSwapОбличчя з референсу. Weight ~1.2–1.4, Stop ~0.98.
PyraCanny / CPDSУтримати позу / структуру тіла.
Stop At / WeightДо якого % і з якою силою діє референс. Advanced — показує всі параметри.
11 · Input Image

Inpaint · Describe · Metadata

Inpaint / OutpaintБіла маска пензлем → перегенерувати зону. Outpaint — розширити кадр. Improve Detail — face/hand. Modify Content — змінити одяг/фон.
Generate maskАвто-маска (SAM, u2net).
DescribeЗавантаж фото → AI пише промпт → потрапляє в Prompt. Навчання формулювань.
Enhance uploadГотове PNG → тільки Enhance, без нової генерації.
MetadataЗавантаж PNG Fooocus → Apply Metadata → відновлює промпт, seed, налаштування.
12 · Enhance

Enhance (галочка + панель)

Другий етап після основної генерації. SAM знаходить зону → inpaint дорисовує деталі.

#1, #2, #3…Окремі проходи. Enable — увімкнути.
Detection promptЩо знайти: face, hand, eyes, body.
Inpaint Enginev2.6 — алгоритм Fooocus.
Denoising Strength0.35–0.45 для face — типово.
Respective FieldРозмір області навколо маски.
Кейс: Generate → Enhance ON → #1 Enable → detection face → Generate знову.
13 · Workflow

Типові сценарії

A. Проста генерація

Advanced ON → Prompt → Generate → Download PNG

B. Краще обличчя

Generate → Enhance ON → #1 face → або Inpaint на обличчя

C. Конкретне обличчя

Input Image → Image Prompt → FaceSwap + референс → Prompt про тіло/сцену

D. Повтор вдалого кадру

Metadata → Apply → змінити seed → Generate

17 · Self-test

Чеклист після Generate

  • Обличчя — очі, streak у волоссі, риси з промпту
  • Тіло / поза — дах, перила, full body
  • Фон — нічне місто, bokeh вогнів
  • Стиль — phone candid, не 3D-рендер
  • Шкіра — pores, imperfections, не пластик
  • Немає blur / watermark

Якщо обличчя слабке (крок 2)

Enhance ON → #1 Enable → detection face → Strength 0.35–0.45 → Generate (той самий seed).

Якщо потрібне конкретне обличчя (крок 3)

Input Image → Image Prompt → FaceSwap → Weight 1.3, Stop 0.98 → той самий Prompt.

Звіт: час до PNG ___ хв · якість 1–5 ___ · Colab впав так/ні · seed + скрін при багах.
18 · Roadmap

Куди можна розвивати Fooocus Pro

Зараз — безкоштовний Colab T4 (MVP, тест, демо). Якщо потрібні швидкість, стабільність і вища якість — є три напрямки масштабування.

Варіант A

Локально на ПК

Якщо є NVIDIA GPU (рекомендовано від 8 GB VRAM, комфортно 12 GB+).

  • Клон foocus_sunside → run.bat / entry_with_update.py
  • Той самий preset + прапорці (--disable-censor)
  • Без обривів Colab-сесії
  • Можна увімкнути Enhance / більше Image Number без страху timeout
💰 Разова вартість GPU · найкращий контроль
Варіант B

Colab Pro / Pro+

Платна підписка Google — той самий ноутбук, кращі умови.

  • Швидші GPU (частіше A100 / кращий T4 tier)
  • Довші сесії, менше «відвалів» посеред генерації
  • Менше черг на GPU
  • Мінімальні зміни в продукті — лише акаунт Colab
💰 ~$10–50/міс · найпростіший апгрейд
Варіант C

Хмара з оплатою за годину

Перенос Fooocus на RunPod, Vast.ai, Lambda, Paperspace тощо.

  • RTX 4090 / A100 — швидше і стабільніше за free T4
  • Docker або VM + Gradio з --share або власний URL
  • Передбачуваний uptime для команди / продакшн-use
  • Можна один інстанс на кілька користувачів (VPN / auth)
💰 Pay-as-you-go · для регулярного production

Покращення продукту (не лише GPU)

Preset «phone selfie»Окремий рецепт під селфі — менше ручних стилів.
Prompt Builder у UIФорма замість довгого англ. тексту — персонаж, сцена, камера.
Pro mode lite для ColabAuto Enhance face без падінь на T4.
Власний hosted UIОдин лінк для команди без Colab — потребує GPU-сервер (варіант C).
Нові моделіFlux / SD3 — окремий Colab, коли стабілізується стек (не зараз).
Для PM: free Colab = zero cost MVP. Colab Pro = low-friction upgrade. Локально / RunPod = коли якість і uptime важливіші за простоту входу.
